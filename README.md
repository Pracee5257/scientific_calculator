<h1><u><b>Scientific Calculator with Graphical Representation</b></u></h1>

A command line based scientific calculator structured with Python, able to conduct mathematical operations, data visualization creating graphical maps and error handling.


<h1><u><b>Project Summary</b></u></h1>

This multifunctional calculator is designed to operate basic to advanced level algorithms. In addition, it can demonstrate graphical visualization of functions. These features of this calculator are highly beneficial for learners to have not only mathematical knowledge but also programming skills.

<h1><u><b>Features</b></u></h1>

•	Basic arithmetic operations<br>
•	Trigonometric functions (sin, cos, tan, cosec, sec, cot)<br>
•	Logarithmic functions (log, ln)<br>
•	Graph plotting<br>
•	Mode switching (Degree, Radian)<br>
•	Error handling<br>
•	Input Validation<br>
•	History management<br>
•	User friendly menu bar

<h1><u><b>Technology used</b></u></h1>

•	Python 3.13<br>
•	Mathplotlib<br>
•	NumPy
